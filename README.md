## Packages

`npm install -D @elseu/sdu-react-scripts-rollup`

- [Rollup](./packages/rollup)
### Babel Plugins
`npm install -D @elseu/sdu-react-scripts-babel-plugins`

- [Babel plugins](./packages/babel-plugins)
