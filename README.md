# sdu-react-scripts
Generic scripts and config files for your next (React) project

## Installation

### Eslint config
`npm install -D @elseu/sdu-react-scripts-eslint`
### Rollup config
`npm install -D @elseu/sdu-react-scripts-rollup`
### Babel Plugins
`npm install -D @elseu/sdu-react-scripts-babel-plugins`
