# Stylelint

## Using Stylelint config

For linting your CSS with the `stylelint` CSS linter. Using the Sdu Stylelint configuration is simple: Extend the `@elseu/sdu-react-scripts-prettier` in your `.stylelintrc` file and you're good to go:

> ```json
> {
>  "extends": "@elseu/sdu-react-scripts-stylelint"
> }
> ```
